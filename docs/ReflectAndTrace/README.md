# Reflection and Traceability Report

Source for the **Reflection and Traceability Report** — team reflection on feedback and changes across the project, with explicit traceability (e.g. Rev 0 → Rev 1) for grading and CEAB learning outcomes.

| File | Role |
|------|------|
| `ReflectAndTrace.tex` | Main LaTeX source: changes in response to feedback, lessons learned, traceability to issues/commits, and related reflection prompts. |
| `Makefile` | Builds the PDF when a LaTeX toolchain is available. |

This is distinct from the Development Plan; it focuses on *what changed because of feedback* and *how* that is evidenced in version control and documentation.
