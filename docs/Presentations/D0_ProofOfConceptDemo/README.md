# Proof of Concept (D0) demonstration

Artifacts for the **proof-of-concept** milestone: experiments, baselines, and metrics that justify the technical approach before full integration.

Typical contents in this folder:

| Item | Purpose |
|------|---------|
| `WhisperBaseline.ipynb`, `Wave2Vec2Baseline.ipynb`, `phoneme.ipynb` | Speech / ASR-related experiments and baselines. |
| `PreprocessingMetrics.ipynb`, `Plots_Metrics_of_WER_CER .ipynb` | Data prep and error-rate visualization. |
| `Keeping_Disease_Data.ipynb` | Dataset or cohort handling notes/experiments. |
| [training](training/) | Optional notebook(s) for fine-tuning (e.g. LoRA). |

Formal slide decks are optional at D0; focus is on reproducible notebooks and figures that support the PoC narrative.
