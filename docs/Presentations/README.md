# Presentations and demonstrations

Materials for **milestones demos and public presentations** of VoiceBridge, organized by deliverable stage. Each stage has its own folder with a short README describing what belongs there.

| Folder | Typical use |
|--------|-------------|
| [D0_ProofOfConceptDemo](D0_ProofOfConceptDemo/) | Proof-of-concept: notebooks, baselines, early metrics, optional training subfolder. |
| [D1_Rev0Demo](D1_Rev0Demo/) | Rev 0 demo: scripts, sample inputs, or supporting files (slides optional). |
| [D2_FinalPresentation](D2_FinalPresentation/) | Final presentation slides and speaker notes. |
| [D3_EXPO](D3_EXPO/) | Expo poster or equivalent (`expo.pdf` when used). |
