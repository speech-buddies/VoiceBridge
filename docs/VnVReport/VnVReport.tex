\subsection{test-PF-1: ASR Latency Benchmark}

\paragraph{Type}
Automated, Dynamic

\paragraph{Test Case Derivation}
PRF-13.1

\paragraph{Test Summary}
The ASR latency benchmark was executed using 20 audio samples spoken by various speakers. For each sample, latency was measured from the marked end-of-speech point to the time at which the transcript became available. The measured latencies were recorded and averaged to evaluate performance against the requirement threshold.

\label{tab:asr-latency-benchmark}

\paragraph{Calculation of Average Latency}
The average latency is calculated as:
\[
\text{Average Latency} = \frac{\sum_{i=1}^{20} L_i}{20} = 5.66 \text{ s}
\]
where \(L_i\) is the latency measured for sample \(i\).

\paragraph{Actual Result}
The measured average latency across the 20 audio samples was 5.66 seconds.

\paragraph{Conclusion}
This test \textbf{does not fully satisfy} PRF-13.1, since the measured average latency exceeded the required threshold of 5 seconds.

\subsubsection{test-PF-2: Command Pipeline Latency}

\paragraph{Type}
Automated, Dynamic

\paragraph{Test Case Derivation}
PRF-13.2

\paragraph{Test Summary}
The command pipeline latency benchmark was run using the same 20 audio samples from the ASR latency test. For each sample, latency was measured from the point at which the transcript became available to the point at which the requested command result was completed by the system. The measured latencies were recorded and averaged to assess compliance with the performance requirement.

\paragraph{Calculation of Average Latency}
\[
\begin{aligned}
\text{Average Latency} &= \frac{1}{20}(8.92 + 9.41 + 12.87 + 14.35 + 9.76 + 13.42 + 15.91 + 12.26 + 11.83 + 13.74 \\
&\quad + 12.95 + 18.62 + 10.54 + 14.08 + 9.88 + 13.67 + 12.43 + 9.15 + 11.96 + 8.64) \\
&= 12.52 \text{ s}
\end{aligned}
\]

\paragraph{Actual Result}
The measured average command pipeline latency across the 20 audio samples was 12.52 seconds.

\paragraph{Conclusion}
This test satisfied the performance requirement, since the measured average latency was below the 15 second threshold.

\subsubsection{test-PF-3: Accuracy \& Precision (Stationary Noise)}

\paragraph{Type}
Offline Evaluation

\paragraph{Test Case Derivation}
PRF-13.1, PRF-13.2

\paragraph{Test Summary}
This test was planned to evaluate ASR and command classification performance under stationary background noise conditions, including AC and fan noise mixed at 20 dB, 10 dB, and 5 dB SNR. The intended evaluation metrics were word recognition accuracy and command precision under noisy conditions.

\paragraph{Result}
This test was not executed during the current development cycle.

\paragraph{Rationale}
Robustness to stationary background noise was identified as a stretch goal for the project. At this stage, development and validation efforts were prioritized toward core functionality under standard operating conditions. As a result, noise robustness testing was deferred and is outside the current project scope.

\paragraph{Conclusion}
No result is reported for this test. Evaluation of performance under stationary noise remains future work.
