# Module Interface Specification (SoftDetailedDes)

Source for the **Module Interface Specification (MIS)** — detailed module-level design: interfaces, state, and behavior for VoiceBridge components.

| File | Role |
|------|------|
| `MIS.tex` | Main LaTeX source for the MIS document. |
| `Makefile` | Builds the PDF (typically `MIS.pdf`) when a LaTeX toolchain is available. |

Auxiliary files produced during compilation (`.aux`, `.log`, `.toc`, etc.) are normal LaTeX artifacts.
