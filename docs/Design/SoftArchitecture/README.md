# Module Guide (SoftArchitecture)

Source for the **Module Guide (MG)** — the software architecture document for VoiceBridge.

| File | Role |
|------|------|
| `MG.tex` | Main LaTeX source: system overview, module decomposition, uses relation, and traceability (including links to the SRS where configured). |
| `Makefile` | Builds the PDF (typically `MG.pdf`) when a LaTeX toolchain is available. |

Auxiliary build outputs (`MG.aux`, `MG.log`, `.bbl`, etc.) are generated by the compiler and can be ignored or cleaned per your workflow.
